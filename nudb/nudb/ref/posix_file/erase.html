posix_file::erase

Remove a file from the file system. 

static
void
erase(
    path_type const& path,
    error_code& ec);

It is not an error to attempt erasing a file that does not exist.

Parameters

path

The path of the file to remove.

ec

Set to the error, if any occurred.
