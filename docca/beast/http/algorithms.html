Algorithms

In addition to the universal message model, Beast provides synchronous algorithms which operate on HTTP/1 messages:

  • read: Parse a message from a stream
  • write: Serialize a message into its wire format on a stream

Asynchronous versions of these algorithms are also available:

  • async_read: Parse a message from a stream
  • async_write: Serialize a message into its wire format on a stream
