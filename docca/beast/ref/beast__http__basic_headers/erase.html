http::basic_headers::erase

Remove a field.

Synopsis 
std::size_t
erase(
    boost::string_ref const& name);
Description

If more than one field with the specified name exists, all matching fields will be removed.

Parameters

Name

Description

name

The name of the field(s) to remove.

Return Value

The number of fields removed.
