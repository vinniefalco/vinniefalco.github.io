http::basic_headers::replace (2 of 2 overloads)

Replace a field value.

Synopsis 
template<
    class T>
std::enable_if< ! std::is_constructible< boost::string_ref, T >::value >::type
replace(
    boost::string_ref const& name,
    T const& value);
Description

First removes any values with matching field names, then inserts the new field value.

Parameters

Name

Description

name

The name of the field

value

The value of the field. The object will be converted to a string using boost::lexical_cast.
