websocket::stream::write_frame (1 of 2 overloads)

Send a message frame on the stream.

Synopsis 
template<
    class ConstBufferSequence>
void
write_frame(
    bool fin,
    ConstBufferSequence const& buffers);
Description

This function is used to write a frame to the stream. The call will block until one of the following conditions is true:

  • The entire frame is sent.
  • An error occurs.

This operation is implemented in terms of one or more calls to the stream's write_some function.

If this is the beginning of a new message, the message opcode will be set to text or binary as per the current setting of the websocket::message_type option. The actual payload sent may be transformed as per the WebSocket protocol settings.

Parameters

Name

Description

fin

true if this is the last frame in the message.

buffers

One or more buffers containing the frame's payload data.

Exceptions

Type

Thrown On

system_error

Thrown on failure.
