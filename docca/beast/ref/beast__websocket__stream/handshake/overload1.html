websocket::stream::handshake (1 of 2 overloads)

Send a HTTP WebSocket Upgrade request and receive the response.

Synopsis 
void
handshake(
    boost::string_ref const& host,
    boost::string_ref const& resource);
Description

This function is used to synchronously send the WebSocket upgrade HTTP request. The call blocks until one of the following conditions is true:

  • A HTTP request finishes sending and a HTTP response finishes receiving.
  • An error occurs on the stream

This function is implemented in terms of one or more calls to the next layer's read_some and write_some functions.

The operation is successful if the received HTTP response indicates a successful HTTP Upgrade (represented by a Status-Code of 101, "switching protocols").

Parameters

Name

Description

host

The name of the remote host, required by the HTTP protocol.

resource

The requesting URI, which may not be empty, required by the HTTP protocol.

Exceptions

Type

Thrown On

system_error

Thrown on failure.

Example 
websocket::stream<ip::tcp::socket> ws(io_service);
...
try
{
    ws.upgrade("localhost", "/");
}
catch(...)
{
    // An error occurred.
}
