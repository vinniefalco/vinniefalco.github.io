prepare_buffers

Return a trimmed, wrapped buffer sequence.

Synopsis 
template<
    class BufferSequence>
prepared_buffers< BufferSequence >
prepare_buffers(
    std::size_t n,
    BufferSequence const& buffers);
Description

This function returns a new buffer sequence which wraps the provided buffer sequence and efficiently presents a shorter subset of the original list of buffers starting with the first byte of the original sequence.

Parameters

Name

Description

n

The maximum number of bytes in the wrapped sequence. If this is larger than the size of buffers, the wrapped sequence will represent the entire input sequence.

buffers

The buffer sequence to wrap. A copy of the sequence will be made, but ownership of the underlying memory is not transferred.
