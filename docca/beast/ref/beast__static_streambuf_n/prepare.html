static_streambuf_n::prepare

Inherited from static_streambuf. Get a list of buffers that represents the output sequence, with the given size.

Synopsis 
mutable_buffers_type
prepare(
    std::size_t n);
Description
Exceptions

Type

Thrown On

std::length_error

if the size would exceed the limit imposed by the underlying mutable buffer sequence.

Remarks

Buffers representing the input sequence acquired prior to this call remain valid.
