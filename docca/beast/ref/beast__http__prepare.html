http::prepare

Prepare a HTTP/1 message.

Synopsis 
template<
    bool isRequest,
    class Body,
    class Headers,
    class... Options>
void
prepare(
    message_v1< isRequest, Body, Headers >& msg,
    Options&&... options);
Description

This function will adjust the Content-Length, Transfer-Encoding, and Connection headers of the message based on the properties of the body and the options passed in.

Parameters

Name

Description

msg

The message to prepare. The headers may be modified.

options

A list of prepare options.

Header

#include <beast/http/message_v1.hpp>
