basic_streambuf::basic_streambuf (1 of 7 overloads)

Move constructor.

Synopsis 
basic_streambuf(
    basic_streambuf&&);
Description

The new object will have the input sequence of the other stream buffer, and an empty output sequence.

Remarks

After the move, the moved-from object will have an empty input and output sequence, with no internal buffers allocated.
