http::parser_v1::upgrade

Inherited from http::basic_parser_v1. Returns true if the message is an upgrade message. 

bool
upgrade() const;

A value of true indicates that the parser has successfully completed parsing a HTTP upgrade message.

Return Value

true if the message is an upgrade message.
