websocket::teardown_tag

Tag type used to find teardown and async_teardown overloads. 

struct teardown_tag

Overloads of teardown and for user defined types must take a value of type websocket::teardown_tag in the first argument in order to be found by the implementation.

Requirements

Header: beast/websocket/teardown.hpp

Convenience header: beast/websocket.hpp
