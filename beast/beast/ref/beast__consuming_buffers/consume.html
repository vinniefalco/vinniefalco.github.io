consuming_buffers::consume

Remove bytes from the beginning of the sequence.

Synopsis 
void
consume(
    std::size_t amount);
Description
Parameters

Name

Description

amount

The number of bytes to remove. If this is larger than the number of bytes remaining, all the bytes remaining are removed.
