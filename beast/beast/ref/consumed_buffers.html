consumed_buffers

Returns a new, consumed buffer sequence. 

template<
    class BufferSequence>
consuming_buffers< BufferSequence, typename BufferSequence::value_type >
consumed_buffers(
    BufferSequence const& buffers,
    std::size_t n);

This function returns a new buffer sequence which when iterated, efficiently represents the portion of the original buffer sequence with n bytes removed from the beginning.

Copies will be made of the buffer sequence passed, but ownership of the underlying memory is not transferred.

Parameters

buffers

The buffer sequence to consume.

n

The number of bytes to remove from the front. If this is larger than the size of the buffer sequence, an empty buffer sequence is returned.

Requirements
