to_string

Convert a ConstBufferSequence to a std::string. 

template<
    class ConstBufferSequence>
std::string
to_string(
    ConstBufferSequence const& buffers);

This function will convert the octets in a buffer sequence to a string. All octets will be inserted into the resulting string, including null or unprintable characters.

Parameters

buffers

The buffer sequence to convert.

Return Value

A string representing the contents of the input area.

Remarks

This function participates in overload resolution only if the buffers parameter meets the requirements of ConstBufferSequence.
