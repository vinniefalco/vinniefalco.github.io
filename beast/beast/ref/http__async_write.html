http::async_write

Start writing a HTTP message to a stream asynchronously. 

template<
    class AsyncWriteStream,
    bool isRequest,
    class Body,
    class Headers,
    class CompletionToken>
void-or-deduced
async_write(
    AsyncWriteStream& stream,
    message< isRequest, Body, Headers > const& msg,
    CompletionToken && token);
Parameters

stream

The stream to send the message on.

msg

The message to send.

token

The handler to be called when the request completes. Copies will be made of the handler as required. The equivalent function signature of the handler must be: 

 void handler(
    error_code const& error // result of operation
);

Regardless of whether the asynchronous operation completes immediately or not, the handler will not be invoked from within this function. Invocation of the handler will be performed in a manner equivalent to using boost::asio::io_service::post().

Remarks

The message must remain valid at least until the completion handler is called, no copies are made.
