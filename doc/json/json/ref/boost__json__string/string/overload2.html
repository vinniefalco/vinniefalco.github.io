string::string (2 of 12 overloads)

Pilfer constructor.

Synopsis 
string(
    pilfered< string > other);
Description

Constructs the string with the contents of other using pilfer semantics. Ownership of the memory_resource is transferred.

Complexity

Constant.

Remarks

After construction, the pilfered-from string may only be destroyed.

Parameters

Name

Description

other

The string to pilfer

See Also

pilfer Pilfering constructors are described in Valueless Variants Considered Harmful, by Peter Dimov.
