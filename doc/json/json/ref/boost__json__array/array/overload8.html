array::array (8 of 11 overloads)

Pilfer constructor.

Synopsis 
array(
    pilfered< array > other);
Description

The array is constructed by acquiring ownership of the contents of other using pilfer semantics.

Remarks

After construction, the moved-from array may only be destroyed.

Complexity

Constant.

Exception Safety

No-throw guarantee.

Parameters

Name

Description

other

The array to pilfer

See Also

pilfer Pilfering constructors are described in Valueless Variants Considered Harmful, by Peter Dimov.
