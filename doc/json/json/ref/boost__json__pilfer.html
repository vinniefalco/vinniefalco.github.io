pilfer

Indicate that an object t may be pilfered.

Synopsis

Defined in header <boost/json/pilfer.hpp> 

template<
    class T>
auto
pilfer(
    T&& t);
Description

This is used similarly to std::move.

Example

This shows how an instance of T may be pilfer-constructed from another T: 

struct T
{
    T( pilfered<T> );
};

void f()
{
    T t1;
    T t2( pilfer(t1) ); // pilfer-construct

    // At this point, t1 may only be destroyed
}
