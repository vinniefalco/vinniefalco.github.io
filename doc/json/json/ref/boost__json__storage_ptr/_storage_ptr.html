storage_ptr::~storage_ptr

Destructor.

Synopsis 
~storage_ptr();
Description

If the pointer has shared ownership of the resource, the shared ownership is released. If this is the last owned copy, the memory resource is destroyed.

Complexity

Constant.

Exception Safety

No-throw guarantee.
