storage_ptr::make_counted_resource

Return shared ownership of a newly created memory resource.

Synopsis

Defined in header <boost/json/storage_ptr.hpp> 

template<
    class U,
    class... Args>
friend storage_ptr
make_counted_resource(
    Args&&... args);
Description

This function creates a memory resource that uses shared ownership. The lifetime of the memory resource will be extended until the last storage_ptr which points to it is destroyed.

Mandates 
std::is_base_of< memory_resource, T >::value == true
Complexity

Same as T( std::forward<Args>(args)... ).

Exception Safety

Strong guarantee.

Parameters

Name

Description

args

Parameters forwarded to the constructor of T.

Template Parameters

Type

Description

The type of memory resource to create.
