parser::write (2 of 2 overloads)

Parse JSON incrementally.

Synopsis 
std::size_t
write(
    string_view s,
    error_code& ec);
Description

This function parses the JSON in the specified buffer. The parse proceeds from the current state, which is at the beginning of a new JSON or in the middle of the current JSON if any characters were already parsed.

The characters in the buffer are processed starting from the beginning, until one of the following conditions is met:

  • All of the characters in the buffer have been parsed, or
  • A complete JSON is parsed, including any optional trailing whitespace in the buffer, or
  • A parsing error occurs.

The supplied buffer does not need to contain the entire JSON. Subsequent calls can provide more serialized data, allowing JSON to be processed incrementally. The end of the serialized JSON is be indicated by calling finish().

Complexity

Linear in s.size().

Return Value

The number of characters consumed from the input, which may be less than the size provided.

Parameters

Name

Description

s

The characters to parse.

ec

Set to the error, if any occurred.
