parser::finish

Parse JSON incrementally.

Synopsis 
void
finish(
    error_code& ec);
Description

The caller uses this function to inform the parser that there is no more serialized JSON available. If a complete JSON is not available the error is set to indicate failure.

Complexity

Constant.

Parameters

Name

Description

ec

Set to the error, if any occurred.
