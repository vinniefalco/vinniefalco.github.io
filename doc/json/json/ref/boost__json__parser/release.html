parser::release

Return the parsed JSON as a value.

Synopsis 
value
release(
    error_code& ec);
Description

If done() returns true, then the parsed value is returned. Otherwise, the error is set to indicate failure. It is necessary to call reset after calling this function in order to parse another JSON.

Complexity

Constant.

Exception Safety

Strong guarantee. If an exception occurs, the valid operations are reset or destruction.

Return Value

The parsed value. Ownership of this value is transferred to the caller.

Parameters

Name

Description

ec

Set to the error, if any occurred.
