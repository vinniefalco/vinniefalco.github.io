value::emplace_bool

Return a reference to a bool, changing the kind and replacing the contents.

Synopsis 
bool&
emplace_bool();
Description

The value is replaced with a bool initialized to false, destroying the previous contents.

Complexity

Linear in the size of *this.

Exception Safety

No-throw guarantee.
