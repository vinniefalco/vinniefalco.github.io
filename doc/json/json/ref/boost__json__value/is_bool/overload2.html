value::is_bool (2 of 2 overloads)

Return a bool pointer if this is a boolean, else return nullptr

Synopsis 
bool*
is_bool();
Description

If this->kind() == kind::bool_, returns a pointer to the underlying boolean. Otherwise, returns nullptr.

Example

The return value can be used in a logical context: 

if( jv.is_bool() )
    return true;

Or the return value can be used to access the boolean later: 

if( auto p = jv.is_bool() )
    return *p;
Complexity

Constant.

Exception Safety

No-throw guarantee.
