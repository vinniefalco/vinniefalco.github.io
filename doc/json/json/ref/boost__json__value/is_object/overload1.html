value::is_object (1 of 2 overloads)

Return an object pointer if this is an object, else return nullptr

Synopsis 
object const*
is_object() const;
Description

If this->kind() == kind::object, returns a pointer to the underlying object. Otherwise, returns nullptr.

Example

The return value can be used in a logical context: 

if( jv.is_object() )
    return true;

Or the return value can be used to access the array later: 

if( auto p = jv.is_object() )
    return *p;
Complexity

Constant.

Exception Safety

No-throw guarantee.
