value::as_bool (1 of 2 overloads)

Return a reference to the underlying bool, or throw an exception.

Synopsis 
bool&
as_bool();
Description

If is_bool() is true, returns a reference to the underlying bool, otherwise throws an exception.

Complexity

Constant.

Exception Safety

Strong guarantee.

Exceptions

Type

Thrown On

std::invalid_argument

! this->is_bool()
