monotonic_resource::~monotonic_resource

Destructor.

Synopsis 
~monotonic_resource();
Description

Deallocates all the memory owned by this resource.

Effects 
this->release();
Complexity

Linear in the number of deallocations performed.

Exception Safety

No-throw guarantee.
