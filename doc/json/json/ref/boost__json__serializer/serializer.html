serializer::serializer

Default constructor.

Synopsis 
serializer();
Description

This constructs a serializer with no value. The value may be set later by calling reset. If serialization is attempted with no value, the output is as if a null value is serialized.

Complexity

Constant.

Exception Safety

No-throw guarantee.
