make_salt

Return a random salt.

Synopsis 
template<
    class = void>
std::uint64_t
make_salt();
Description

This function will use the system provided random number device to generate a uniformly distributed 64-bit unsigned value suitable for use the salt value in a call to create.

Header

#include <nudb/create.hpp>
