win32_file::open

Open a file.

Synopsis 
void
open(
    file_mode mode,
    path_type const& path,
    error_code& ec);
Description
Requirements

The file must not already be open.

Parameters

Name

Description

mode

The open mode, which must be a valid |9|file_mode.

path

The path of the file to open.

ec

Set to the error, if any occurred.
