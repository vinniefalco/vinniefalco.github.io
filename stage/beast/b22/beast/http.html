Beast offers programmers simple and performant models of HTTP messages and their associated operations including synchronous and asynchronous reading and writing of messages and headers in the HTTP/1 wire format using Boost.Asio.

Note

The following documentation assumes familiarity with both Boost.Asio and the HTTP protocol specification described in rfc7230. Sample code and identifiers mentioned in this section are written as if the following declarations are in effect: 

#include <beast/core.hpp>
#include <beast/http.hpp>
using namespace beast;
using namespace beast::http;
