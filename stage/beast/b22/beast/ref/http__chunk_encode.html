http::chunk_encode

Returns a chunk-encoded ConstBufferSequence. 

template<
    class ConstBufferSequence>
implementation-defined
chunk_encode(
    bool fin,
    ConstBufferSequence const& buffers);

This returns a buffer sequence representing the first chunk of a chunked transfer coded body.

Parameters

fin

true if this is the last chunk.

buffers

The input buffer sequence.

Return Value

A chunk-encoded ConstBufferSequence representing the input.

See Also

rfc7230 section 4.1.3

Requirements

Header: beast/http/chunk_encode.hpp

Convenience header: beast/http.hpp
