http::header_parser_v1::complete

Inherited from http::basic_parser_v1. Returns true if the parse has completed succesfully. 

bool
complete() const;

When the parse has completed successfully, and the semantics of the parsed message indicate that the connection is still active, a subsequent call to write will begin parsing a new message.

Return Value

true If the parsing has completed successfully.
