http::body_what

A value indicating how the parser should treat the body. 

enum body_what

Values

normal

The parser should expect a body, keep reading.

skip

Skip parsing of the body.

upgrade

The message represents an UPGRADE request.

pause

Suspend parsing before reading the body.

This value is returned from the on_header callback in the derived class. It controls what the parser does next in terms of the message body.

Requirements

Header: beast/http/basic_parser_v1.hpp

Convenience header: beast/http.hpp
