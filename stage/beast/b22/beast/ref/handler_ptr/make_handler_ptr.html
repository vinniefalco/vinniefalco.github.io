handler_ptr::make_handler_ptr

Construct a new handler_ptr. 

template<
    class U,
    class Handler,
    class... Args>
friend handler_ptr< U >
make_handler_ptr(
    Handler&& handler,
    Args&&... args);
Parameters

handler

The handler. The allocator associated with the handler will be used to allocate memory for the owned object. This argument will be forwarded to the owned object's constructor.

args

Optional arguments forwarded to the owned object's constructor.

Requirements

Header: beast/core/handler_ptr.hpp
