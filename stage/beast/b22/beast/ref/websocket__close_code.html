websocket::close_code

Close status codes. 

enum close_code

Values

none

used internally to mean "no error"

normal
going_away
protocol_error
unknown_data
no_close

Indicates a received close frame has no close code.

bad_payload
policy_error
too_big
needs_extension
internal_error
service_restart
try_again_later
reserved1
no_status
abnormal
reserved2
last

These codes accompany close frames.
