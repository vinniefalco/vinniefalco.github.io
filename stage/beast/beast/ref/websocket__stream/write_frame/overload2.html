websocket::stream::write_frame (2 of 2 overloads)

Send a frame. 

template<
    class ConstBufferSequence>
void
write_frame(
    bool fin,
    ConstBufferSequence const& buffers,
    error_code& ec);

This function is used to write a frame to a stream. The call will block until one of the following conditions is true: * All of the data in the supplied buffers has been written. * An error occurs. This operation is implemented in terms of one or more calls to the stream's write_some function. The actual payload sent may be transformed as per the WebSocket protocol settings. If this is the beginning of a new message, the message opcode will be set to text or binary as per the current setting of the message_type option.

Parameters

fin

true if this is the last frame in the message.

buffers

One or more buffers containing the frame's payload data.

ec

Set to indicate what error occurred, if any.
