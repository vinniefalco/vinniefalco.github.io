http::basic_fields::insert (2 of 2 overloads)

Insert a field value. 

template<
    class T>
std::enable_if< ! std::is_constructible< boost::string_ref, T >::value >::type
insert(
    boost::string_ref name,
    T const& value);

If a field with the same name already exists, the existing field is untouched and a new field value pair is inserted into the container.

Parameters

name

The name of the field

value

The value of the field. The object will be converted to a string using boost::lexical_cast.
