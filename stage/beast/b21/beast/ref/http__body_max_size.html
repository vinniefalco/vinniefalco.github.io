Body maximum size option.

Name

Description

value

Sets the maximum number of cumulative bytes allowed including all body octets. Octets in chunk-encoded bodies are counted after decoding. A value of zero indicates no limit on the number of body octets.

The default body maximum size for requests is 4MB (four megabytes or 4,194,304 bytes) and unlimited for responses.

Remarks

Objects of this type are used with basic_parser_v1::set_option.
