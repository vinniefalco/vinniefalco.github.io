http::write (3 of 4 overloads)

Write a HTTP/1 message to a stream. 

template<
    class SyncWriteStream,
    bool isRequest,
    class Body,
    class Fields>
void
write(
    SyncWriteStream& stream,
    message< isRequest, Body, Fields > const& msg);

This function is used to write a message to a stream. The call will block until one of the following conditions is true:

  • The entire message is written.
  • An error occurs.

This operation is implemented in terms of one or more calls to the stream's write_some function.

The implementation will automatically perform chunk encoding if the contents of the message indicate that chunk encoding is required. If the semantics of the message indicate that the connection should be closed after the message is sent, the error thrown from this function will be boost::asio::error::eof.

Parameters

stream

The stream to which the data is to be written. The type must support the SyncWriteStream concept.

msg

The message to write.

Exceptions

system_error

Thrown on failure.
