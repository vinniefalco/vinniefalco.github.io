prepare_buffer (2 of 2 overloads)

Return a shortened buffer. 

boost::asio::mutable_buffer
prepare_buffer(
    std::size_t n,
    boost::asio::mutable_buffer buffer);

The returned buffer points to the same memory as the passed buffer, but with a size that is equal to or less than the size of the original buffer.

Parameters

n

The size of the returned buffer.

buffer

The buffer to shorten. Ownership of the underlying memory is not transferred.

Return Value

A new buffer that points to the first n bytes of the original buffer.
